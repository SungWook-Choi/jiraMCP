# 작업 지시서: v0.3.0 4번 조회 / 댓글 입력 분기

## 문서 목적

이 문서는 `qwen-jira-mcp@0.3.0` 추가 개발 항목 중 4번인 CLI 첫 진입 흐름 변경과 댓글 입력 UX의 구현 범위를 고정하기 위한 작업 지시서다.

상위 기준 문서는 `docs/versions/0.3.0/work-instruction.md`다.

## 작업 배경

현재 CLI는 실행 후 바로 Jira 조회 흐름으로 들어간다.

하지만 요구사항은 처음 진입 시 사용 목적을 먼저 나누는 것이다.

사용자는 아래 두 가지 중 하나를 먼저 선택해야 한다.

1. `조회`
2. `댓글 입력`

`조회`를 선택하면 현재 조회 기능을 그대로 사용하고, `댓글 입력`을 선택하면 Jira 이슈에 댓글을 남기는 흐름으로 들어가야 한다.

## 목표

1. CLI 첫 진입 시 `조회 / 댓글 입력` 선택 흐름을 추가한다.
2. `조회` 선택 시 기존 조회 기능을 유지한다.
3. `댓글 입력` 선택 시 `기본 / 주간이슈` 선택 흐름을 추가한다.
4. `주간이슈` 선택 시 댓글 본문 앞에 `[주간 이슈]` prefix를 자동으로 붙인다.
5. 사용자는 이슈를 `이슈명 검색` 또는 `이슈 키 직접 입력`으로 선택할 수 있어야 한다.
6. `이슈명 검색`을 기본 UX로 제공한다.
7. 댓글 본문은 기본 문자열 입력으로 받는다.

## 구현 범위

이번 작업 범위는 아래로 제한한다.

1. CLI 첫 진입 상위 메뉴 추가
2. `댓글 입력` 하위 메뉴 추가
3. `기본 / 주간이슈` 선택 흐름 추가
4. `이슈명 검색 / 이슈 키 직접 입력` 선택 흐름 추가
5. 이슈명 기반 Jira 이슈 검색 기능 추가
6. 선택된 Jira 이슈에 댓글 등록하는 요청 경로 추가
7. `주간이슈` 선택 시 `[주간 이슈]` prefix 자동 부착
8. 전송 전 확인 메시지 추가
9. 관련 문서 또는 사용 예시 정리

## 세부 규칙

1. CLI 첫 질문은 `조회` 또는 `댓글 입력`이다.
2. `조회`를 선택하면 기존 조회 흐름을 그대로 사용한다.
3. `댓글 입력`을 선택하면 다음 단계에서 `기본` 또는 `주간이슈`를 선택하게 한다.
4. `기본`은 사용자가 입력한 문자열을 그대로 댓글 본문으로 사용한다.
5. `주간이슈`는 사용자가 입력한 문자열 앞에 `[주간 이슈]`와 줄바꿈을 자동으로 붙인다.
6. `댓글 입력`에서 이슈 선택은 아래 두 가지를 제공한다.
7. `1. 이슈명 검색`
8. `2. 이슈 키 직접 입력`
9. 기본 추천 흐름은 `이슈명 검색`이다.
10. `이슈명 검색`은 제목 일부 입력으로 결과 목록을 보여주고, 사용자가 번호로 선택하게 한다.
11. `이슈 키 직접 입력`은 고급 사용자용 빠른 경로로 제공한다.
12. 댓글 전송 전에는 최종 확인 단계를 둔다.

## 이슈 선택 기준

이번 작업에서 이슈 선택 UX는 아래 기준으로 본다.

1. 사용자는 Jira 이슈 키를 모를 수 있다.
2. 따라서 `이슈 키 직접 입력만 강제`하는 방식은 채택하지 않는다.
3. `이슈명 검색 후 선택`을 기본 UX로 채택한다.
4. 검색 결과가 여러 건이면 번호 선택 방식으로 정한다.
5. `이슈 키 직접 입력`은 보조 옵션으로 유지한다.

## 댓글 내용 기준

이번 작업에서 댓글은 기본 문자열로 받는다.

예시:

```text
작업 중 이런 문제가 있습니다
```

`주간이슈` 선택 시 실제 저장 본문 예시는 아래와 같다.

```text
[주간 이슈]
작업 중 이런 문제가 있습니다
```

## 제외 범위

아래 항목은 이번 작업 범위에서 제외한다.

1. 멘션 기능
2. Jira 사용자 검색과 `accountId` 기반 멘션 ADF 구성

4. 다중 줄 편집기나 외부 에디터 연동
5. 댓글 수정 / 삭제 기능
6. Web UI 댓글 입력 기능
7. MCP에서 댓글 입력 기능
8. `[주간 이슈]` 댓글 조회 기능 자체 구현

## 멘션 관련 판단

멘션은 Jira에서 단순 문자열 `@이름`만으로 안정적으로 처리되지 않을 수 있다.

일반적으로 사용자 검색, `accountId`, Jira 댓글 본문의 ADF 구조가 필요하므로 이번 작업 범위에서 제외한다.

이번 단계에서는 문자열 댓글 등록에 집중한다.
