# React Rule

## Encoding

- React 관련 문서와 코드는 모두 `UTF-8` 기준으로 읽는다.
- React 관련 문서와 코드는 모두 `UTF-8` 기준으로 저장한다.

## 목적

이 문서는 추후 Web UI가 필요할 때 참고하는 React 규칙이다.

## 현재 상태

- 현재 MVP 범위에는 Web UI 구현이 포함되지 않는다.
- React 작업이 시작되기 전까지는 참고 문서로만 유지한다.

## 후속 적용 원칙

1. Web UI는 기존 NestJS 코어와 API 위에 얇게 붙인다.
2. Jira 조회와 비즈니스 로직은 프론트가 아니라 백엔드 코어가 담당한다.
3. React 구현이 시작되면 별도 작업지시서 범위를 먼저 확정한다.
